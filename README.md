# *⋆˚✰ ݁˖⭑. Instructions *⋆˚✰ ݁˖⭑.

To be able to play the game you have to build the executable file with every dependency (here MinilibX and my own libft, which I have included) :

```bash
make
```

And then run it with a .ber map, I have provided some in my maps/ folder, you can make your own or you can use a [map generator](https://github.com/ELmounikor/42-So_long-random-map-generator). 

```bash
./so_long maps/map1.ber
```
